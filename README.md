# VS Code Minimal Setup

A clean and minimal Visual Studio Code configuration focused on productivity and distraction-free coding.

This setup removes most UI elements and keeps only what is necessary for writing code.

---

## Features

* Minimal UI
* Hidden activity bar
* Hidden status bar
* Single tab editor
* Clean file explorer
* No minimap
* No breadcrumbs
* Reduced visual noise
* Custom cursor style
* Smooth caret animation

---

## Theme

**Best Themes - Andromeda Mariana**

---

## Font

**Dank Mono**

Settings used:

```
editor.fontFamily: Dank Mono
editor.fontSize: 14
editor.fontLigatures: true
editor.lineHeight: 2.5
```

---

## UI Tweaks

This setup includes several interface tweaks:

* Hidden activity bar
* Sidebar moved to the right
* Hidden status bar
* Hidden scrollbars
* Hidden editor decorations
* Minimal suggestion UI
* Disabled visual distractions

---

## Explorer Configuration

* Compact folders disabled
* Drag and drop confirmations disabled
* File decorations disabled

---

## Editor Behavior

* Relative line numbers
* Tab size: 2
* Trailing whitespace removed automatically
* Final newline inserted automatically
* Snippet suggestions prioritized

---

## Extensions Used

Recommended extensions:

* Material Icon Theme
* Custom UI Style
* Best Themes

---

## Installation

1. Clone this repository

```
git clone https://github.com/YOUR_USERNAME/vscode-setup.git
```

2. Copy the settings file to your VS Code configuration folder.

### Windows

```
C:\Users\YOUR_USER\AppData\Roaming\Code\User
```

3. Replace or merge the `settings.json` file.
